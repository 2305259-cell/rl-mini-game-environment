# RL Mini Game Environment

This project implements a simple Reinforcement Learning mini-game environment.

The agent moves inside a 3x3 grid and must reach the goal while avoiding obstacles.

Symbols used in the grid:

A = Agent  
G = Goal  
X = Obstacle  
. = Empty Cell

## Files

environment.py - Game environment  
main.py - Run the full simulation  
demo.py - Demo script  
requirements.txt - Dependencies  

## How to Run

Install requirements:

pip install -r requirements.txt

Run demo:

python demo.py

Run full simulation:

python main.py
